# PixiJS Template

## Description

This is a Pixel Builder Template using [PixiJS](https://pixijs.com/)

This allow for 2D Projects

- Sprites
- Text
- Graphics
- ...

## Getting Started

```sh
pixel create pixi-project -t pixi
cd pixi-project
yarn dev
```
