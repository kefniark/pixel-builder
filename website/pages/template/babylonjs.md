# BabylonJS Template

## Description

This is a Pixel Builder Template using [BabylonJS](https://www.babylonjs.com/)

This allow for 3D Projects

- 3D Meshes
- Lightning
- Shaders
- ...

## Getting Started

```sh
pixel create babylon-project -t babylon
cd babylon-project
yarn dev
```
